# Tugas Sinkronisasi Komputasi Terdistribusi - Kelompok 1

## Setup

```bash
git clone https://github.com/prayudahlah/sinkronisasi-komter-kelompok-1
cd sinkronisasi-komter-kelompok-1
uv sync
```

## Cara Menjalankan

Server:
```bash
uv run server.py
```

Client:
```bash
uv run client.py --server prayudahlah:8000
```
